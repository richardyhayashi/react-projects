## Create React App

`$ npm create vite@latest {.|project-name} -- --template react`

### Edit 'vite.config.js'

`export default defineConfig({`
`  ...`
`  server: {`
`    host: "0.0.0.0",`
`    port: 3000,`
`  }`
`  ...`
`});`

### Start App

`$ npm run dev`

## Install React-Icons

`$ npm install react-icons --save`
